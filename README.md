# Party Pooper Jeeps

Party Pooper Jeeps is a retro-inspired split-screen multiplayer jeep game where players compete by invading each other's base and returning presents and the stolen Christmas tree to their own base. The game includes various mechanics such as fuel, speed, score, collecting presents and stealing the Christmas tree.

## Technologies

- **JavaScript** & **TypeScript**
- **Phaser.js**
- **React**
- **Bolt.new** (for initial setup and idea generation)

This project started on **bolt.new** and was further refined using my knowledge of **JavaScript, TypeScript, and Phaser**.

## Installation

1. Clone the repository:
   ```bash
   git clone <repo-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd party-pooper-jeeps
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the game locally:
   ```bash
   npm run dev
   ```
5. Open the game in your browser via `http://localhost:3000`

## Known Bugs

- **Quit Menu not always works in Single Player** sometimes the Quit Menu isn't correctly working in Single Player. Refresh the browser to play again.
- **Audio:** Audio playback does not always work correctly.
